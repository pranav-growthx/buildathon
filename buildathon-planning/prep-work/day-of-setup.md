---

### Common Setup Issues & Solutions

**Issue 1: Git not installed**
- **Quick fix:** Send to git-scm.com/downloads
- **Time:** 10-15 mins
- **Backup:** They can commit later, start coding first

**Issue 2: Cursor/Windsurf not installed**
- **Quick fix:** cursor.sh or codeium.com/windsurf
- **Time:** 5-10 mins
- **Backup:** Use VS Code with Copilot temporarily

**Issue 3: Node.js not installed**
- **Quick fix:** nodejs.org/en/download
- **Time:** 10 mins
- **Backup:** Use Python/FastAPI stack instead

**Issue 4: No Firebase/Supabase account**
- **Quick fix:** Create during ideation phase (10am-12pm)
- **Not blocking:** Accounts can be created later

**Issue 5: No project idea**
- **Response:** "That's fine, we're brainstorming at 10am. Think about what annoys you."
